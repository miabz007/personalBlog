# Personal Portfolio Website

## Overview

This project is a personal portfolio website developed using HTML, CSS, Bootstrap, and JavaScript. It showcases my profile, skills, projects, work experience, and contact information through a responsive multi-page design.

## Features

* Responsive navigation bar
* Home page introduction
* Work and services section
* Portfolio gallery
* Resume and skills display
* About Me page
* Contact page
* Mobile-friendly design

## Technologies Used

* HTML5
* CSS3
* Bootstrap
* JavaScript
* Font Awesome

## Installation

1. Download or clone the project.
2. Open the project folder.
3. Launch `index.html` in a web browser.
